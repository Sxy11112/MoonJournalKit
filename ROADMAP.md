# Roadmap

## 0.1

- MWAL v1 记录格式与 CRC32C
- 事务恢复、检查点和分段规划
- 断电切断与位翻转验证
- CLI、基准、稳定 JSON 和多后端 CI

## 0.2

- 流式增量解码器，避免整段日志驻留内存
- 可配置事务内存上限与超限诊断
- 分段清单编码和原子替换协议示例
- 属性测试生成器与更大规模随机崩溃矩阵

## 0.3

- 独立 I/O 适配包，覆盖 Native 文件和浏览器持久化
- 组提交、同步策略和延迟统计接口
- 格式兼容性语料库与跨版本迁移工具

路线图中的 I/O 能力会保持在独立子包，核心格式与恢复器继续维持零第三方依赖。
